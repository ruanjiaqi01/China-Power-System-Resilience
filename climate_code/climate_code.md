# climate_code
## Overview
The code in this folder deals with the processing of cmip and era weather data.
## Code description
### net_step1.py
Save the nc file as a mat file and an npy file.
### province_step2.py
Call the inverse geocoding api to get the province for each latitude and longitude.
### change_step3.py
Divide Inner Mongolia into East Inner Mongolia and West Inner Mongolia.
### uv2wind_step4.py
Converts u and v wind speeds to total wind speed.
### mat2csv_step5.py
Convert latitude/longitude climate data to csv data by province.
### wind_simulate_step9.py
Simulate future weather data for each node based on historical data.
### wind_branch_simulate_step10.py
Simulate future branch weather data based on historical data.
### false_step11.py
Calculate the failure rate based on node wind speed
### branch_false_step12.py
Calculate the failure rate based on line wind speed
### false_is_step13.py
Determine if a line has failed
### load_output_step15.py"
Calculate the effect of temperature on load demand
### load_times_step16.py
Fine tune load to hourly and impose volatility
### P_output_step17.py
Calculate the power generated by each node
### false_time_step18.py
Calculate drop time per failure
